## Technical Details

### How QR Code is Generated:

```javascript
// 1. Load QR library from CDN
<script src="https://cdn.jsdelivr.net/npm/qrcodejs@1.0.0/qrcode.min.js">

// 2. When video ends, generate QR code
new QRCode(element, {
    text: 'https://www.linkedin.com/in/gaston-d-859653184/',
    width: 200,
    height: 200,
    colorDark: '#00ffff',  // Cyan (Data Bleed theme)
    colorLight: '#000000'  // Black background
});

// 3. QR code appears as an image element
// 4. Styled with CSS (border, glow, pulse)
```

### The QR Code Contains:
- Your LinkedIn URL (encoded in the pattern)
- Error correction (works even if partially obscured)
- Optimized for phone camera scanning

## Troubleshooting

### QR Code Doesn't Appear:
1. Check browser console for errors
2. Verify internet connection (CDN required)
3. Wait 10 seconds (fallback timeout)
4. Check if video file exists and plays

### QR Code Not Scannable:
1. Ensure good lighting
2. Hold phone steady
3. Try different camera apps
4. Click QR code to open link directly

### Glitch Effect Too Intense:
1. Edit `js/qr-logo-system.js`
2. Reduce animation values in `logoToQRGlitch`
3. Shorten duration from 1s to 0.5s
